About This Project

Campus cleanliness analysis for faster review.

This interface lets users capture or upload images, preview them, and run inference to support waste and cleanliness inspection workflows.

Purpose

The project is designed to help review campus waste conditions through an image-based interface that highlights detected trash and returns an annotated result.

How It Works

Users submit an image, the model processes it through the inference API, and the result card is updated with an annotated preview plus the final prediction label.

Team

Analog Minds by Farhan Haroon and Shambhavi Pandey, with the Perception and Intelligence Lab, EE, IIT Kanpur context reflected in the current interface.

Deployment Shape

The frontend is a static GitHub Pages-style interface, while inference is handled separately through the connected prediction API.
