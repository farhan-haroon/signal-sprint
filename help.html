How to use the interface.

A quick reference for uploading, previewing, and running inference on images.

Quick Steps

  1. Use `Take a photo` or `Upload images` on the home page.
  2. Confirm the preview cards appear in the results area.
  3. Click `Run inference` to process the queued images.
  4. Review the annotated outputs and prediction labels.

Troubleshooting

  • If no preview appears, make sure the selected file is an image.
  • If inference fails, verify that the backend API endpoint is active.
  • If camera capture does not open, try a supported mobile browser.

Notes

The interface is static, but inference depends on a live external API. If the model endpoint is unavailable, previews will still show while result processing will fail.
